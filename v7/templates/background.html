Original Proposal

Motivation

Time series data are ubiquitous in various real-world applications, ranging from finance to healthcare. Traditional methods such as ARIMA, LSTM, etc., have shown their effectiveness in domain specific applications. Recently, Large Language Models (LLMs) has shown its great power in various nature language related tasks, e.g., Q&A, machine translation, standard tests (e.g., SAT, GRE, etc.). Since time series data and language data share many common properties, there is a significant potential in applying existing large language models to time series analysis. This project aims to explore potential solutions to apply and compare existing LLMs to different time series analysis tasks.

Requirements

  1. Download and deploy a LLM (e.g., GPT2/GPT3/Llama 2/3) on a GPU server/cloud environment. Perform standard language inference task based on existing LLMs.
  2. Develop an interface that a user can upload time series data (such as stock prices, weather data etc. ), preprocess, tokenize the data, and showcase the results.
  3. Utilize a large language model to obtain representations of the time series data, interpret the results, and explore useful patterns.
  4. Design algorithm/prompting mechanism so that LLM can perform classification tasks.
  5. Design algorithm/prompting mechanism so that LLM can perform forecasting tasks.
  6. Compare different pre-trained LLM models.
  7. Try to explore the explainability of LLMs.
  8. Try to finetune the LLM if possible.
  9. Provide analytics dashboards that display findings and patterns for time series data.
